# SEO-based Product Description Generator

This project is an automated SEO-oriented product description generation system designed for e-commerce platforms (e.g. Trendyol).

## Features
- CSV-based product ingestion
- HTML output fully compatible with Trendyol description format
- SEO-oriented content generation using LLM
- Banned-word filtering
- Duplicate sentence detection
- Minimum / maximum character enforcement
- Versioned generation history (PASS / FAIL)
